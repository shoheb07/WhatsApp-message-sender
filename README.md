# WhatsApp Message Sender

## Project Overview

The WhatsApp Message Sender is a Python automation tool that sends scheduled WhatsApp messages using WhatsApp Web.

The project uses the pywhatkit library for browser automation and messaging.


# Features

- Send scheduled WhatsApp messages
- Instant message support
- WhatsApp Web integration
- Lightweight automation project


# Technologies Used

- Python
- pywhatkit
- WhatsApp Web


# Installation

Install pywhatkit:

pip install pywhatkit

# Run Project

python whatsapp_sender.py


# Requirements

- Internet connection
- Logged-in WhatsApp Web account
- Browser installed
